# De Vuelta

App hiperlocal para reunir mascotas perdidas con sus dueños y coordinar rescate de animales callejeros en la Alcaldía Benito Juárez, CDMX.

Proyecto desarrollado por [Unico Studios](https://github.com/unicostudios-mx).

## Estado
🚧 En desarrollo — Fase 0 (Fundamentos)

## Stack
- Next.js 15 (App Router) + TypeScript + Tailwind
- Supabase (Auth, Postgres + PostGIS, Storage)
- Mapbox GL JS
- OneSignal (push notifications)
- Anthropic Claude API (visión para matching)
- Vercel (deploy)
